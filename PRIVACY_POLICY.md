# Privacy Policy — HLTV Fantasy Hard Mode

Short summary

HLTV Fantasy Hard Mode modifies the display of HLTV (hltv.org) fantasy pages in your browser to hide booster and trigger-rate UI elements. The extension uses local browser storage only to remember your on/off preference. The extension does not collect, transmit, or store any personal or browsing data to external servers.

Data collected

- Local storage: a single boolean flag that records whether the user enabled the hiding feature. This is stored using the browser's `chrome.storage` API.
- No other data is collected, persisted, or transmitted.

How data is used

- The stored preference determines whether the content script should hide UI elements on HLTV pages. All modifications occur locally within the user's browser.

Third parties

- This extension does not send any data to third parties. It modifies page appearance only and performs no external network communication.

Source code

- The full source code is available at: https://github.com/amritparmanand/hltv-fantasy-hardmode
